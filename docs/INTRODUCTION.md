# Understanding the project scope

To this project here is the scope:
0. Collection of data
0. Annotation of data
0. Multiple sub-programs that handle: 
1. Creating synthetic data with all the quirks of real data
2. Managing the existence and properties of all data in a database 
3. Benchmarking models and record keeping of performance in a db         
0. A front-end that neatly wraps all the above work as another Good Notes, Obsidian alternative.
